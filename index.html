Les branches dans git

Créer une branch = git branch < nomDeBranch >

Voir les branches créer = git branch

Basculer sur l'autre branch = git checkout < nomDeBranch >

Pour fusionner deux branches = git merge < nomDeBranch >

Pour supprimer une branch vide = git branch -d < nomDeBranch>

Pour supprimer une branch non vide = git branch -D < nomDeBranch >

Vhost et divers ligne de commande


Procédure d'installation d'un Vhost

Se trouver dans le dossier cd/etc/apache2/sites-available/

Taper sudo -s pour être en root

cp pour copier et renommer= cp copier => nomDuFichier.conf renommer => nomDuFichier.conf

Taper nano < nomDuFichier.conf >

Faire ctrl + w et ctrl +r ensuite taper le nom du fichier à renommer et le nom du fichier renommer

Enregistrer avec ctrl + o

Retourner dans le dossier /etc et taper nano hosts

Taper la ligne 127.0.0.1 + nom du fichier

Dans /etc taper a2ensite < nom du fichier.conf >

Taper systemctl reload apache2

Quitter le mode root

Allez dans cd~/dossier en question

Taper systemctl restart apache2
