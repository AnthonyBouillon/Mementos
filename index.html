exit : pour se retirer du root

rm -rf (nomdudossier): supprimer un dossier (attention)

touch : crée un fichier

mkdir : créer un ou plusieur dossier cote a cote

mkdir -p crée un dossier et sous dossier

mv : déplacer

root : sudo -s

apt upgrade : mise a jour

nano nomDuFichier : Ouvrir un fichier

shift ctrl m = affichage portable sur navigateur

ctrl + f pour rechercher une lettre ou mot

Ligne de commande pour l'affichage

Afficher le manuel pour la commande ls : man ls

Afficher le contenu du dossier courant: ls

Afficher le contenu du dossier courant avec les fichiers cachés: ls -a

Afficher le contenu du dossier courant version longue sans les fichiers cachés: ls -l

Afficher le contenu du dossier courant version longue avec les fichiers cachés : ls -Al

Afficher le chemin du dossier courant : pwd

Se déplacer dans le dossier Document : cd Documents

Afficher le contenu de ce dossier : ls

Se mettre en super utilisateur : sudo -s

Mettre à jour la liste des paquets : apt update ( apt-get)

Mettre à jour les paquets : apt upgrade

Trouver tous les paquets contenant le mot XML : apt search xtml

Installer le paquet git : apt install git

Créer un dossier website dans le dossier Document : mkdir website

Créer un dossier et un sous dossier en une seule ligne de commande dans le dossier website : Memento/assets : mkdir -p Memento/assets

Se déplacer dans le dossier nouvellement créé et faire trois nouveaux dossiers en une seule ligne de commande : css, img, js : mkdir css img js

Dans le dossier Memento créer le fichier style.css : touch style.css

Pour rentrer dans le fichier : nano index.html

Déplacer le fichier style.css dans le dossier css du dossier assets : mv style.css assets/css/
