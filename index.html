Procédure d'installation d'un vhost

  1. Se trouver dans le dossier cd/etc/apache2/sites-available/
  2. cp pour copier et renommer: cp copier => exercice1 renommer => exercie2
  3. Taper nano + nom du fichier
  4. Faire ctrl + w et ctrl +r ensuite taper le nom du fichier à renommer et le nom du fichier renommer
  5. Enregistrer avec ctrl + o
  6. Retourner dans le dossier /etc et taper nano hosts
  7. Taper la ligne 127.0.0.1 + nom du fichier
  8. Dans /etc taper a2ensite + nom du fichier.conf
  9. Taper systemctl reload apache2
  10. Quitter le mode root
  11. Allez dans cd~/www/dossier en question
  12. Taper systemctl restart apache2
