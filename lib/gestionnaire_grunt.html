Grunt

Il sert à minifier vos css, js

npm init

npm install grunt --save-dev : Installer grunt en local

touch Gruntfile.js : Créer le fichier de configuration

Remplir le fichier en fesant un copier coller

npm install grunt-contrib-cssmin --save-dev : Installer le plugin css minifier

npm install grunt-contrib-uglify --save-dev : Installer le plugin js minifier

Gruntfile.js : remplir totalement le fichier

grunt css et grunt js : Effectué la minification
