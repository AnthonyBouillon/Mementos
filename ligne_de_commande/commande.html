Ligne de commande divers sous ubuntu

ctrl + u : vérifier sont code sur le navigateur

ctrl + maj + m : Site format portable vue par le navigateur

ctrl + f: pour rechercher une lettre ou un mot


firefox : lancer firefox

gedit : bloc note


cd : Entrer dans un dossier (cd /etc)

cd .. : revenir dans le dossier d'avant

pwd : Afficher le chemin du dossier courant


sudo -s : Être en super utilisateur

exit : pour se retirer du root


rm -rf (nomdudossier): supprimer un dossier (attention)

touch : crée un fichier

mkdir : créer un ou plusieur dossier

mkdir -p: crée un dossier et sous dossier

mv : déplacer


dpkg -i nomdufichier : installer un exe

apt update : Mettre à jour la liste des paquets apt upgrade : Mettre à jour les paquets apt install git : Installer le paquet git
